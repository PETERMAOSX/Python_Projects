* 基本操作

  * git init  --初始化仓库

  * git status --检查仓库的状态

  * git add --向暂存区中添加文件

  * git commit --保存仓库的历史记录

  * git remote add origin https://github.com/PETERMAOSX/Temp.git

  * git branch -M main

  * git push -u origin main
